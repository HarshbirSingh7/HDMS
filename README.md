The **Hotel Database Management System (HDMS)** is a **GUI-based Python application** that performs CRUD operations (Create, Read, Update, Delete) on hotel guest records.  
It uses a CSV file (`Hotel.csv`) for persistent data storage and follows a modular structure by separating frontend, backend, and data visualization logic.

This project is especially useful for:
- Students learning **Python GUI programming**
- Beginners exploring **database concepts without SQL**
- Understanding **file handling and data visualization**
